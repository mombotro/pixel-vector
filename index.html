Keyboard Shortcuts

  • 1-6 Select drawing tools
  • V Select tool (move/edit)
  • F Fill cell tool
  • Right-click Add/remove nodes (select mode)
  • Ctrl+C Copy selected shape
  • Ctrl+V Paste shape
  • Del/Backspace Delete shape
  • Ctrl+Z Undo last shape
  • Ctrl+S Save to file
  • H Lock horizontal
  • G Toggle grid display
  • O Toggle outline mode
  • + Zoom in
  • - Zoom out
  • 0 Reset zoom
  • ESC Cancel current operation
